# SPOTIFY-AUTHENTICATION-CLONE

A Spotify-inspired authentication interface built using **HTML**, **CSS**, and **JavaScript**. This project features interactive Login and Create Account popups with client-side form validation and a modern Spotify-themed user interface.

## FEATURES

- Spotify-inspired UI design
- Login popup modal
- Create Account popup modal
- Username validation
- Password validation
- Background blur effect when forms are opened
- Responsive layout
- Smooth hover animations and transitions
- Clean and organized code structure

## TECHNOLOGIES USED

- HTML
- CSS
- JavaScript

## VALIDATION RULE

### USERNAME

- Maximum 15 characters
- Allows:

  * A-Z
  * a-z
  * 0-9
  * Underscore (_)
- No spaces allowed

### PASSWORD

- Must contain exactly 6 numeric digits
